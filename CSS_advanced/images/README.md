# Advanced CSS Project

## Description
This project is an exploration of advanced CSS concepts including selectors, properties, values, the box model, and grid systems using floats. The goal is to build a functional and visually appealing homepage for "Techium" while ensuring W3C compliance.

## Learning Objectives
* Understand the difference between block and inline styling.
* Implement a CSS reset for browser consistency.
* Utilize CSS variables for maintainable code.
* Work with various CSS layout techniques (Grids, Floats).
* Master CSS animations and transforms.

## Requirements
* Editors: vi, vim, emacs, VSCode.
* Browser: Chrome (version 78.x).
* W3C compliant code.
